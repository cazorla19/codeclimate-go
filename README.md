# codeclimate-go

A client written in [go](https://golang.org/) for provisioning [Codeclimate](https://codeclimate.com)

### Usage

```go
go get -u "github.com/babbel/codeclimate-go/codeclimate"
```

```go
import "github.com/babbel/codeclimate-go/codeclimate"
```

### Running the tests

```bash
go test -race -v ./...
```
